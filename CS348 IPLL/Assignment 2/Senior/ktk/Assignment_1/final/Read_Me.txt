Assumptions and Remarks in Assignment 1:
1. Make sure to have a file named "input.txt" with the input assembly code in the same directory as the assembler.
2. Label, Opcode and Operand each have a field size of exactly 10 characters (Don't use tabs for padding).
3. Three files "intermediate.txt", "listing.txt" and "output.txt" will be generated at the end of the execution. 
4. If there is any error it will just be printed on the terminal.
5. Comments must have "." symbol in the Label field.
6. The hexadecimal constants of type X'' are assumed to be of even number of digits.
7. The program name is assumed to be atmost 6 characters long.
8. There are nospaces before and after ',' while using indirect addressing

Command to run: g++ -g assembler.cpp
